# P3 Priors — prior-year research used as priors for P4

Scripts and analyses from **Prosperity 3** (last year's competition), preserved here as a starting point for P4 work.

## What's here

| File | Purpose |
|---|---|
| [`manuals.py`](./manuals.py) | P3 manual-challenge solver(s) — used as a structural template when approaching P4 manuals. |
| [`fresh_scan_r2.py`](./fresh_scan_r2.py) | Signal scan rebuilt from scratch against P3 R2 data — used as a practice run before P4 opened. |
| [`r2_sensitivity.sh`](./r2_sensitivity.sh) | Shell script wrapping the P3 R2 sensitivity sweep. |

## Why a separate folder

These files predate the P4 competition. They're not part of any P4 shipped strategy — they're the *priors* I formed before P4 round 1 opened. Separating them by name (`p3_*`) and folder is one of the small organizational moves I'd make differently next year: the prior-year material is genuinely useful but shouldn't sit inline with the current-year code.
